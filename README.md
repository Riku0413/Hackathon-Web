# Hackathon-Web

## 概要
自作したナレッジベース（Webアプリ）のフロントエンド

## 実行方法

1. リポジトリのクローン
   ```
   git pull "url"
   ```
   
2. モジュールのインストール
   ```
   npm install react-scripts
   ```

3. アプリの起動
   ```
   npm start
   ```

## 搭載機能

バックエンドのリポジトリに示す

## ファイル構成

```
src
├── auth-components
├── curriculum-components
├── Header-components
├── http-components
├── Item-components
├── Page-components
└── Post-components
```

## 技術スタック

- React（TypeScript）
- Material UI
- Firebase（認証）
- Vercel（過去のデプロイ時に使用）
